# 📂 FS-QR (File Sharing & QR) 🚀

こんにちは！ FS-QR へようこそ！👋
このアプリは、誰でも簡単に**ファイル共有**や**メモの共有**ができる便利なツールです。
QRコードを使ってスマホにファイルを送ったり、みんなで一つのノートを編集したりできます✨

---

## ✨ 何ができるの？

- 📱 **QRでサクッと共有**
  ファイルをアップロードすると QRコード が出てくるので、スマホで読み取るだけでダウンロードできます！

- 🏠 **グループでファイル共有**
  「合言葉」を決めて部屋を作れば、みんなでファイルをアップしたりダウンロードしたりできます。

- 📝 **みんなでノート編集**
  リアルタイムで更新されるノート機能！みんなでアイデアを書き込んだり、メモを残したりできます。

---

## 🚀 使い方 (はじめかた)

パソコンでこのアプリを動かすための手順です。難しくないですよ！💪

### 1️⃣ 準備するもの
- **Docker** (これが入っていればOK！)

### 2️⃣ 設定ファイルを作る
プロジェクトの中に `.env` という名前のファイルを作って、パスワードなどを設定します。
(サンプルとして適当な文字を入れていますが、実際は好きなパスワードに変えてね！)

```env
SQL_HOST=db
SQL_USER=user
SQL_PW=password
SQL_DB=fsqr
SECRET_KEY=secret
ADMIN_KEY=admin
MANAGEMENT_PASSWORD=manage
```

### 3️⃣ アプリを起動する
ターミナル(黒い画面)を開いて、以下のコマンドを入力してエンターキーを押すだけ！

```bash
docker-compose up --build
```

### 4️⃣ ブラウザで開く
準備ができたら、ブラウザで以下のURLにアクセスしてください。

👉 `http://localhost:5000`

これで完了です！🎉
