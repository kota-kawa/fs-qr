PCやスマートフォンの画面をそのまま動画として記録する「画面録画（スクリーンレコーディング）」。かつては専門の録画ソフトが必要でしたが、現在では主要なOSすべてに「標準機能」として無料で搭載されています。

操作手順のマニュアル作成、遠隔地にいるメンバーへのバグレポート、オンラインミーティングのアーカイブ、ゲームプレイの保存など、画面録画が活躍するシーンは非常に多くあります。本記事では、Windows、Mac、iPhone、Androidのそれぞれにおいて、余計なアプリをインストールすることなく無料で画面録画を行う手順と、録画後に発生しがちな「動画ファイルが重くて送れない」という問題を解消する共有テクニックを解説します。

1. Windowsで画面録画する方法（標準機能）

Windows 10およびWindows 11には、標準で2つの録画方法があります。

方法A：Xbox Game Bar（手軽でおすすめ）

主にゲームプレイ用ですが、ブラウザやExcelなど一般的なアプリの録画にも使えます。（※ただし、デスクトップ全体やエクスプローラーは録画できません）

  1. 録画したいアプリのウィンドウをアクティブにします。
  2. キーボードの「Windowsキー + G」を同時に押すと、Game Barのメニュー画面が開きます。
  3. キャプチャウィンドウの「録画ボタン（丸アイコン）」をクリックするか、ショートカットキー「Windowsキー + Alt + R」を押すと、即座に録画が始まります。
  4. もう一度同じ操作をするか、画面上の停止ボタンを押すと録画が終了し、ビデオフォルダ内の「キャプチャ」フォルダにMP4形式で保存されます。

方法B：Snipping Tool（Windows 11のみ）

Windows 11の「Snipping Tool」は動画撮影にも対応し、デスクトップの特定範囲を指定して録画できます。

  1. 「Windowsキー + Shift + S」（またはSnipping Toolを検索して起動）を押します。
  2. 上部のメニューから「ビデオ（カメラアイコン）」を選択します。
  3. 「新規」をクリックし、画面上の録画したい範囲をドラッグで囲みます。
  4. 「スタート」をクリックするとカウントダウン後に録画が始まります。

2. Macで画面録画する方法（標準機能）

Macは「スクリーンショット」ツールバーを使って、デスクトップ全体または一部を簡単に録画できます。

  1. キーボードの「command + shift + 5」を同時に押します。
  2. 画面下部にツールバーが表示されます。「画面全体を収録」または「選択部分を収録」のいずれかのアイコンをクリックします。
  3. 「収録」をクリックすると録画がスタートします。
  4. メニューバーに表示される「停止ボタン（丸の中に四角のアイコン）」をクリックして終了します。デフォルトではデスクトップに動画（QuickTimeムービー）が保存されます。

3. iPhone / iPadで画面録画する方法

iOSでは、コントロールセンターから素早く画面収録を開始できます。あらかじめ設定をしておく必要があります。

事前準備：

「設定」アプリ ＞ 「コントロールセンター」を開き、「コントロールを追加」リストの中から「画面収録」をタップして追加しておきます。

録画手順：

  1. 画面の右上から下へスワイプ（ホームボタンがある機種は下から上へスワイプ）してコントロールセンターを開きます。
  2. 「二重の丸いアイコン（画面収録ボタン）」をタップします。
  3. 3秒のカウントダウン後に録画が始まります（画面上部の時計部分が赤くなります）。
  4. 終了するには、赤い時計部分をタップするか、再度コントロールセンターを開いてボタンをタップし、「停止」を選択します。動画は「写真」アプリに保存されます。

4. Androidで画面録画する方法

Android 11以降には「スクリーンレコード」機能が標準搭載されています。

  1. 画面を上から下に2回スワイプして「クイック設定パネル」を開きます。
  2. 「スクリーンレコード」（または「画面録画」）のアイコンをタップします。（見つからない場合は、パネルの編集マークからボタンを追加してください）
  3. 音声の録音有無や画面上のタップを表示するかを設定し、「開始」をタップします。
  4. カウントダウン後に録画が始まります。録画を止めるには、通知バーを引き下げて「停止」をタップします。動画は「フォト」や「ギャラリー」アプリのライブラリに保存されます。

録画した重い動画ファイルをスムーズに共有するコツ

画面録画した動画ファイルは、数分撮影しただけでも数十MB〜数百MBに達することがあります。メールやビジネスチャット（SlackやChatworkなど）で送ろうとすると、容量制限に引っかかったり、送信・受信側の双方でダウンロードに非常に長い時間がかかってしまいます。

こうした大容量動画の受け渡しには、アカウント登録不要で今すぐ使える無料ファイル共有サービス「FS!QR」が大変便利です。

ブラウザでFS!QRを開き、録画した動画ファイルをドラッグ＆ドロップしてアップロードするだけで、相手に伝えるためのダウンロードURLとQRコードが瞬時に発行されます。相手がスマホならQRコードをカメラでスキャンするだけで再生・ダウンロード画面に進めるため、PCからスマホへの動画移動も一瞬で終わります。

パスワードによるアクセス保護や、一定時間後の自動データ消去機能を備えているため、企業の社外秘のデモ画面や操作手順などを共有する際にも安心です。重い動画共有のストレスをゼロにするFS!QRを、画面録画とセットでぜひ活用してください。
